# UniGrade Calculator 

A C# console application designed to calculate a student's cumulative GPA and letter grade based on their subject scores.

## How It Works

The program interacts with the user through the console to perform the following:
1. Prompts the user to enter the total number of subjects.
2. Collects the numerical grade for each subject iteratively.
3. Calculates the GPA based on a standard point system (>= 90 is 4.0, >= 85 is 3.7, etc.).
4. Evaluates and displays the final calculated GPA along with the equivalent letter rating (A, B+, B, C, D, or F).

## Tech Stack

- C#
- .NET Console Application
- Basic Control Flow (If-conditions, Loops)

## Getting Started

To run this project locally:
1. Clone the repository to your local machine.
2. Open the project using Visual Studio or any standard C# IDE.
3. Run the application and follow the on-screen console instructions.
